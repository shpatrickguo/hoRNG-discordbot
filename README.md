# hoRNG-discordbot
Discord bot for hoRNG server.

#Installations
```
npm i dotenv
npm i discord.js
npm i fs
npm install ms
npm install @discordjs/opus ffmpeg-static yt-search ytdl-core 
npm i weather-js
```
[FFmpeg](https://ffmpeg.org/download.html)
